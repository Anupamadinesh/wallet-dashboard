# 💰 Wallet Dashboard Application

A responsive and modular wallet dashboard built using React and Node.js. This application allows users to manage wallet balances, perform transactions, and view transaction history with real-time API integration.

---

## 🚀 Features

### 🔹 Frontend (React)
- Responsive dashboard UI
- Reusable components (cards, lists, modals)
- Wallet balance display
- Transaction history UI
- Interactive modals for deposit, withdraw, and adding bank accounts

### 🔹 Backend (Node.js + Express)
- REST APIs for wallet operations
- Deposit & Withdraw functionality
- Transaction history management
- Bank account management

### 🔹 API Integration
- Frontend connected to backend APIs using Axios
- Real-time updates after transactions
- Proper state management using React hooks

---

## 🛠️ Tech Stack

- **Frontend:** React, CSS
- **Backend:** Node.js, Express
- **API Calls:** Axios
- **Version Control:** Git & GitHub
- **Deployment:** Vercel (Frontend), Render (Backend)
